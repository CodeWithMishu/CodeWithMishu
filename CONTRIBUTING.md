# Contributing to CodeWithMishu

Thanks for your interest in contributing. This repository contains my GitHub profile README and the automation that keeps it updated. Contributions are welcome in the form of suggestions, bug reports, and improvements.

## How to Contribute

1. Fork the repository
2. Create a new branch for your change
3. Make your edits
4. Submit a pull request with a clear description of what you changed and why

## What You Can Contribute

- **Bug fixes** -- broken links, typos, incorrect references, SVG issues
- **Workflow improvements** -- better GitHub Actions configurations, performance, error handling
- **Design suggestions** -- color palette ideas, layout improvements, new widget recommendations
- **Documentation** -- clearer setup instructions, better inline comments

## Guidelines

- Keep the dark theme consistent (background `#0B1220`, accent `#38BDF8`)
- Test that any external service URL you add is currently working
- Keep the README scannable -- use tables and short text blocks
- Do not add skill progress bars or other subjective rating systems
- Keep workflow files minimal and well-commented

## Reporting Issues

If you find a broken widget, dead link, or rendering issue, open an issue with:

- What is broken
- Where it appears (section name or line number)
- Suggested fix if you have one
